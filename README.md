<h1> - Rock, Paper & Scissors</h1>

<p>
This is a simple implementation of the classic <b>Rock, Paper & Scissors</b> game built using HTML, CSS, and JavaScript.
</p>

<p>
The main goal of this project is to demonstrate <b>basic JavaScript understanding</b> from scratch. 
It focuses on fundamental concepts rather than complexity.
</p>

<hr>

<h2> - Project Objective</h2>

<ul>
  <li>Understand how JavaScript interacts with the DOM</li>
  <li>Handle user input using click events</li>
  <li>Build simple game logic using conditions</li>
  <li>Update UI dynamically</li>
</ul>

<hr>

<h2> - How It Works</h2>

<ol>
  <li>User selects Rock, Paper, or Scissors</li>
  <li>Computer generates a random choice</li>
  <li>Winner is decided using basic conditions</li>
  <li>Result is displayed on screen</li>
</ol>

<hr>

<h2> - Concepts Used</h2>

<ul>
  <li>DOM Manipulation </li>
  <li>Event Listeners</li>
  <li>Functions</li>
  <li>Conditional Statements </li>
  <li>Random Function</li>
</ul>

<hr>

<h2> - Project Structure</h2>

<pre>
index.html   → Game structure  
styles.css   → Styling  
script.js    → Game logic  
</pre>
